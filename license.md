### How to Build Your First Airship

1. Craft **Ship Anchor** (Iron Ingot + Copper + Wood)
2. Place anchor in world (where you want ship center)
3. Right-click anchor → **"Enter Build Mode"**
4. Build ship using any blocks within the wireframe
5. Add **Balloon Blocks** for lift (hydrogen or helium)
6. Add **Engine Blocks** for movement
7. Add **Ship Helm** for control
8. Exit Build Mode → Right-click anchor → **"Assemble"**
9. Right-click Ship Helm to pilot!

### How to Balance Lift vs Weight

1. Open Helm UI → **"Ship Stats"**
2. Check **Total Weight** vs **Total Lift**
3. Lift must exceed weight by 10-20% for stable flight
4. Add more balloon blocks if overweight
5. Use **Ballast Control** slider for fine adjustment
6. Tip: Place balloons symmetrically (avoids tipping)

### How to Install Engines

1. Craft **Engine Block** (steel casing + gears + copper)
2. Place on ship (attached to anchor grid)
3. Connect power source:
   - Steam: water tank + furnace heater
   - Combustion: gasoline tank + throttle
   - Electric: RF battery connection
4. Configure **Thrust Direction** (sneak + right-click)
5. Arrow shows direction (forward, backward, up, down)
6. Use multiple engines for stronger thrust

### How to Add Weapons

1. Craft **Weapon Mount** (steel + iron + copper)
2. Place weapon on mount:
   - **Cannon** (basic)
   - **Missile Pod** (advanced)
   - **Laser** (tech)
3. Craft **Weapon Station** (gold + redstone + iron)
4. Place near weapon (within 5 blocks)
5. Right-click station to control weapon
6. Crosshair appears — aim and fire!

### How to Use Blueprints

1. Build ship you like
2. Right-click anchor → **"Save Blueprint"**
3. Name your design (e.g., "Cargo Hauler v1")
4. Blueprint saved to `blueprints/` folder
5. To load: Place new anchor → **"Load Blueprint"**
6. Select saved design → **"Build from Blueprint"**
7. Resources are consumed during build

### How to Set Up Multiplayer Crew

1. Pilot enters Ship Helm
2. Other players right-click any seat/weapon station
3. **Assign Roles** via Helm UI:
   - Gunner (controls weapons)
   - Engineer (repairs damage)
   - Navigator (sets waypoints)
4. Ship can carry unlimited players
5. All players see shared ship health

## 💡 Pro Tips

- **Start small** — Build a 10x10 ship first to learn physics
- **Symmetrical balloons** — Prevents listing to one side
- **Use hydrogen for heavy ships** — More lift but dangerous
- **Place engines at center of mass** — Better turning
- **Add a keel** — Low-weight blocks on bottom prevents flipping
- **Test in creative first** — Before building in survival
